# Workforce Scheduling Automation System

Sistema desenvolvido em **Python** para geração automática de escalas operacionais com interface gráfica.

O projeto foi criado para automatizar processos que antes eram feitos manualmente em planilhas, reduzindo erros e aumentando a eficiência na organização de turnos de trabalho.

---

## 📌 Visão Geral

Este sistema permite cadastrar colaboradores, definir cidades ou unidades operacionais e gerar automaticamente escalas de trabalho dentro de um período selecionado.

A aplicação possui interface gráfica construída com **CustomTkinter**, permitindo que usuários sem conhecimento técnico utilizem o sistema.

As escalas geradas podem ser exportadas para **Excel** ou **HTML editável**, facilitando a distribuição e ajustes posteriores.

---

## 🚀 Funcionalidades

* Cadastro de colaboradores
* Remoção e atualização de colaboradores
* Geração automática de escalas por período
* Distribuição de turnos e folgas
* Suporte para múltiplas cidades ou unidades
* Exportação da escala para Excel
* Exportação da escala para HTML editável
* Interface gráfica moderna
* Armazenamento local de dados em JSON

## ⚙️ Instalação

### 1. Clonar o repositório

```
git clone https://github.com/SEU_USUARIO/workforce-scheduling-automation.git
```

### 2. Acessar a pasta do projeto

```
cd workforce-scheduling-automation
```

### 3. Instalar dependências

```
pip install -r requirements.txt
```

### 4. Executar o sistema

```
python Lscale.py
```
